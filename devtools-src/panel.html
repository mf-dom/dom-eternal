What is XSS?

XSS attacks are a type of injection, in which malicious scripts are injected into otherwise trusted websites.

So what's DOM Eternal?

DOM Eternal identifies a specific type of XSS vulnerability that occurs when the DOM (Document Object Model) is mutated by a Javascript function.

So, how do I use it?

  1. On your desired webpage, open Chrome Devtools (a.k.a. right click in the webpage and click inspect element)
  2. Start recording in DOM Eternal
  3. Select the functions that you want to test for XSS vulnerabilities
  4. Click the dice to start the fuzzer and wait for the results!
